# Relational Database Queries

## Project Overview
This project demonstrates SQL querying techniques used for business intelligence and relational database reporting. The goal was to retrieve, clean, aggregate, and analyze sales data using structured SQL queries.

## Skills Demonstrated
- SELECT queries
- WHERE filtering
- UPDATE statements
- GROUP BY aggregation
- ORDER BY sorting
- SQL joins
- Aggregate functions
- Reporting query design
- Relational database analysis

## Project Highlights
This project includes SQL queries for identifying and correcting inconsistent data, calculating total sales by product, filtering customer purchases, analyzing average purchases by region, and joining supplier data with sales records.
